Fix the `LRUCache` class in `src/lru_cache.py`.

The cache must evict the least recently used key when it reaches capacity.

Rules:
- `get(key)` must return the stored value and mark the key as most recently used;
- `put(key, value)` must update existing keys without duplicating them in the usage order;
- when inserting a new key into a full cache, evict the least recently used key;
- `get(key, default)` must return `default` for missing keys.
